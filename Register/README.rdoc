= Registreringsapplikation

Applikation för att registrera applikationer som vill använda Webb-API'et som ska skapas.

==Instruktioner för att köra applikationen
* Ladda ned applikationen och öppna den i din Rails miljö.
* Kör en <code>bundle install</code> i terminal-fönstret.
* Kör <code>rails s</code> för att starta servern.
* Öppna applikationen i en webbläsare.
